# GenieClient Community Standards

Welcome to the official community standards repository for GenieClient.

GenieClient is an open-source MUD client for DragonRealms on play.net/dr.

GenieClient is an independent community-driven project and is not affiliated with, endorsed by, or operated by Simutronics or play.net.

This repository contains shared governance, contribution guidelines, issue templates, and community standards used across the GenieClient organization.

## Goals

- Keep GenieClient open source and community driven
- Encourage contributions upstream instead of long-term forks
- Maintain a welcoming and collaborative development environment
- Support plugin and map contributors
- Preserve long-term stability and maintainability

## Contributing

We welcome:
- Bug reports
- Pull requests
- Plugin improvements
- Mapping updates
- Documentation improvements
- Community discussions

Forks are welcome for experimentation, but long-term improvements should be proposed back to the official GenieClient repositories whenever possible.

## Organization Repositories

Core repositories may include:
- Genie4
- Maps
- Plugins
- SDK and tooling repositories
